# Oh My Cover Design

用 Claude Code + AI 图片生成模型，做出高质量的公众号/小红书封面。

10 种构图风格可选，逐步问答收集需求，输出可直接投喂图片生成模型的提示词。

---

## 效果

这个 skill 会：

1. 读取你的文章内容
2. 逐步问你 8 个问题（风格、人物、表情、素材、配色、字体、字体颜色、标题）
3. 输出一段精准的提示词，拿去跑图即可

---

## 10 种构图风格

| 风格 | 特点 |
|------|------|
| 深色渐变风 | 人物居中，大字覆盖后方，高对比强冲击 |
| 纯色扁平风 | 人物抠图感，纯色背景，干净清爽 |
| 产品主视觉风 | UI截图/产品占主体，人物做引导手势 |
| 对比卡片风 | 人物手持两张对比卡片，适合前后对比内容 |
| 极简留白风 | 大面积留白，文字是主视觉锤 |
| 海报拼贴风 | 多张参考图叠加，分层有纵深 |
| 人物侧置留白风 | 人物偏一侧，大面积留给标题 |
| 背影构图风 | 人物背对镜头，制造代入感 |
| 局部出镜风 | 只露手/半脸/侧脸，产品是主角 |
| 正面对视风 | 直视镜头，文字环绕脸部 |

## 安装方法

### 前提

- 已安装 [Claude Code](https://claude.ai/code)

### 步骤

1. 下载 `SKILL.md`，放到你本地的 skills 目录：

```bash
mkdir -p ~/.claude/skills/cover-design-open
cp SKILL.md ~/.claude/skills/cover-design-open/SKILL.md
```

2. 在 Claude Code 中，输入 `/cover-design-open` 启动 skill，或者直接把文章内容发给 Claude 并说「帮我设计封面」

---

## 使用方法

把你的文章内容发给 Claude，skill 会依次问你：

1. **选风格** — 10 种风格选一种
2. **图1** — 上传人脸参考图，或描述人物外貌
3. **人物表情** — 8 种表情选一种
4. **额外素材** — 有没有图2、图3（UI截图、产品图等）
5. **背景色调** — 浅色/深色/暖/冷/撞色/交给模型
6. **字体风格** — 黑体/圆体/手写/无衬线/宋体/交给模型
7. **字体颜色效果** — 白/黑/渐变/描边/交给模型
8. **封面标题** — 自动生成候选，你来确认或修改

最后输出提示词，投喂图片生成模型即可。

---

## 多参考图支持

- **图1**：固定为人脸参考图，保持五官一致性
- **图2 起**：UI截图、产品图、其他素材，skill 会把它们融入构图
