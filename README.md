# Oh My Cover Design

把文章扔给 agent，它替你设计封面、想标题、写提示词。你只需要回答几个问题，然后拿提示词去跑图。

支持 Claude Code、Codex 及任何支持自定义 skill 的 AI agent。

## 10 种构图风格

深色渐变风 / 纯色扁平风 / 产品主视觉风 / 对比卡片风 / 极简留白风 / 海报拼贴风 / 人物侧置留白风 / 背影构图风 / 局部出镜风 / 正面对视风

每种风格都有独立的构图逻辑和提示词模板，不会撞款。

---

## 安装

```bash
git clone https://github.com/feitangyuan/oh-my-cover-design.git \
  ~/.claude/skills/cover-design-open
```

装好之后把文章发给 agent，skill 自动触发。
