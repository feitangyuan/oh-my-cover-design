# Rule Candidates

Candidate rules live here before they are promoted into references.

Promotion threshold:

- 3 similar observations: candidate rule.
- 5-10 stable observations: promote to references.
- Strong expert prior: candidate immediately, but mark evidence as expert prior.

## Candidate Template

```text
## Candidate: [rule name]

Evidence:
- [observation]
- [observation]
- [observation]

Rule:
[candidate rule]

Status: observing
```
